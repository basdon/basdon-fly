echo
----
Relays in-game chat and IRC chat.
This is an anna module, see github.com/yugecin/anna

configuration
-------------
echo.channel            -- channel that is used to echo (#basdon.echo)

commands
--------
echo                    -- check if the socket is alive
players                 -- check who is online ingame
echo-ping               -- sends a ping to server and sends a message when a
                           pong is received
